# WannaCry勒索病毒检测查杀与安全加固工具

## Description

**WannaCry勒索病毒**可加密硬盘文件，受害者必须支付高额赎金才有可能解密恢复。本工具用于WannaCry勒索病毒的检测、查杀与安全加固。
